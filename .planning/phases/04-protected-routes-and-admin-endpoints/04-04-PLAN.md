<task type="auto">
  <name>Task 1: GET /admin/users — list all users with counts</name>
  <files>app/api/routes/admin.py, app/main.py, tests/test_admin_routes.py</files>
  <action>
- Create app/api/routes/admin.py with APIRouter(). GET /admin/users (or prefix /admin, path /users): Depends(require_admin). Query all User; for each user compute orders_count (count Order where user_email == user.email or join Order on Order.user_email == User.email) and active_commemoration_count (count Commemoration join Order where Order.user_email == user.email and Commemoration.is_active == True and expires_at >= today). Use single query with scalar_subquery or two subqueries in select to avoid N+1. Response: list of { id, email, role, is_active, created_at (isoformat), orders_count, active_commemoration_count }. Register in main.py: include_router(admin.router, prefix="/api/v1", tags=["admin"]). In admin.py use APIRouter(prefix="/admin") and @router.get("/users") so the full path is /api/v1/admin/users.
- Add tests: no token -> 401; user token -> 403; admin token -> 200 with list containing orders_count and active_commemoration_count (ADMN-02).
  </action>
  <verify>
    <automated>docker compose run --rm api pytest tests/test_admin_routes.py -v -k "list or users or get" -x</automated>
